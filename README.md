# Employee Salary Sheet

## Description
This is a simple Python program that calculates an employee's salary details.

The program takes:
- Employee Name
- Employee Code
- Basic Salary

It calculates:
- DA (Dearness Allowance)
- TA (Travel Allowance)
- HRA (House Rent Allowance)
- Gross Salary
- PF (Provident Fund)
- Net Salary

## Features
- User input based program
- Performs salary calculations automatically
- Displays complete employee salary details

## Requirements
- Python 3.x
